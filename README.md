## Project Overview
This project predicts price change categories based on historical data using a Random Forest model for multiclass classification.

## Features and Target
- **Features:** Derived from 8 technical indicators.
- **Target Classes:**
  - `>1.5 stdev`
  - `0.5 to 1.5 stdev`
  - `-0.5 to 0.5 stdev`
  - `-1.5 to -0.5 stdev`
  - `<-1.5 stdev`
